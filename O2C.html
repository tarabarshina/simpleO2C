📝 Obsidian → Confluence マークアップ変換ツール

✅ 対応している機能

  • ✔️ 見出し (# → h1. **text**)
  • ✔️ 太字 (**text** → *text*)
  • ✔️ 斜体 (*text* → _text_)
  • ✔️ 打ち消し線 (~~text~~ → -text-)
  • ✔️ テーブル (Confluence形式に変換)
  • ✔️ Callouts → {panel} マクロ（色付き）
  • ✔️ Quote (> text → {quote})
  • ✔️ ネストされたリスト (# と * でレベル表示)
  • ✔️ コードブロック (``` → {code})
  • ✔️ インラインコード (` → {{}})
  • ✔️ リンク ([text](url) → [text|url])

使い方: Confluenceのエディタで「マークアップを挿入」を選択し、変換されたテキストを貼り付けてください。
